# MoodMash Documentation

Welcome to the MoodMash documentation. This guide will help you understand the architecture, features, and APIs of the MoodMash application.

## What is MoodMash?

MoodMash is a secure, internationalized messaging application that allows users to share their moods and communicate with end-to-end encryption. It features support for multiple authentication methods, including WebAuthn/Passkeys, and supports 20+ languages including right-to-left languages.

## Getting Started

To get started with MoodMash:

1. **Installation**
   ```bash
   git clone https://github.com/yourusername/moodmash.git
   cd moodmash
   npm install
   npm run dev
   ```

2. **Configuration**
   Set up your environment variables according to the `.env.example` file.

3. **Database Setup**
   ```bash
   npx prisma migrate dev
   npx prisma generate
   ```

## Key Features

- **End-to-End Encryption**: Secure messaging with client-side encryption
- **Multi-Factor Authentication**: Enhanced security with MFA options
- **Internationalization**: Support for 20+ languages including RTL support
- **WebAuthn/Passkeys**: Modern, passwordless authentication

## Architecture

MoodMash follows a clean architecture approach:

- **Frontend**: React components with Tailwind CSS styling
- **Backend**: Next.js API routes
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js for user authentication
- **State Management**: Combination of Jotai, Zustand, and React Query
