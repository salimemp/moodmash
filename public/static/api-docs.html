Base URL

https://moodmash.win/api

All API requests should be made to this base URL.

Authentication

Creating an API Token

  1. Log in to your MoodMash account
  2. Go to Settings → API Tokens
  3. Click "Create New Token"
  4. Name your token and set permissions
  5. Copy the token (you won't see it again!)

Using Your Token

Include your API token in the Authorization header:

curl -H "Authorization: Bearer moodmash_user_xxx" \
     https://moodmash.win/api/moods

Moods API

POST /moods

Create a new mood entry

{
  "emotion": "happy",
  "intensity": 4,
  "notes": "Had a great day!",
  "logged_at": "2025-11-22T12:00:00Z"
}
GET /moods

Get your mood entries

Query parameters: limit, emotion, start_date, end_date

GET /stats

Get mood statistics and analytics

Query parameters: days (default: 30)

Files API

POST /files/upload

Upload a file (avatar, mood image, voice note)

Content-Type: multipart/form-data

curl -X POST \
  -H "Authorization: Bearer moodmash_user_xxx" \
  -F "file=@avatar.jpg" \
  -F "type=avatar" \
  https://moodmash.win/api/files/upload
GET /files

List your uploaded files

GET /files/:key

Download a file by its key

Rate Limits

User Tokens

  • 100 requests per hour
  • 1,000 requests per day

Account Tokens

  • 1,000 requests per hour
  • 10,000 requests per day

Rate limit info is included in response headers: 

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1732291200

Response Codes

200 OK - Request successful
201 Created - Resource created successfully
400 Bad Request - Invalid request format
401 Unauthorized - Invalid or missing token
429 Too Many Requests - Rate limit exceeded
500 Server Error - Something went wrong
